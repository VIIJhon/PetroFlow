# Petroflow Scripts

This directory contains helper scripts organized into subfolders:

- `setup/` — Scripts for database migrations, seeding, and environment setup.
- `testing/` — Test helpers and verification scripts used in CI and local checks.
- `utilities/` — Maintenance utilities such as backups and admin helpers.
- `examples/` — Example scripts and simulators to demonstrate features.

Run PowerShell scripts on Windows and Python scripts with the project virtual environment activated.
